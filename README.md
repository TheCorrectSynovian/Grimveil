# Grimveil

**CURSED AS HELL**

Grimveil is a Minecraft resource and datapack that transforms your game into a horrifying experience. This pack is intentionally designed to look and feel unsettling, making it perfect for those who enjoy a spooky and cursed aesthetic.

## About
This pack was created using **Faithful** as a base. Many textures from the Faithful pack may still be present in Grimveil. Please refer to the [LICENSE.txt](LICENSE.txt) file for more details regarding the usage of assets.

## Features
- A cursed and unsettling visual style.
- Spooky advancements and messages to enhance the horror theme.
- Compatibility with Minecraft 1.21.11.

## Disclaimer
This pack is intentionally designed to look awful and cursed. If you enjoy a polished and clean aesthetic, this pack may not be for you.
